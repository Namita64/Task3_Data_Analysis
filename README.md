# Task 3 - Sales Data Analysis Project

## 📌 Project Description
This project performs data analysis on a CSV file containing product sales data using Python and Pandas.

It calculates key statistics like total sales, average sales, highest sales, and lowest sales.

---

## 🚀 Features
- Reads data from CSV file
- Uses Pandas for data analysis
- Computes total, average, max, and min values

---

## 🛠️ Technologies Used
- Python
- Pandas library
- CSV file handling

---

## ▶️ How to Run
Step 1: Install pandas
pip install pandas

Step 2: Run the program
python main.py

---

## 📊 Output Example
Total Sales = 202000  
Average Sales = 20200  
Highest Sales = 50000  
Lowest Sales = 10000
